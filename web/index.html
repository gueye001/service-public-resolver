À propos

Comment fonctionne l'agent

  1. Embedding — la question est encodée avec BGE-M3 (1024 dimensions).
  2. Recherche FAISS — les 5 questions officielles les plus proches (similarité cosinus) sont récupérées.
  3. Reranking — un CrossEncoder réordonne finement les 5 candidats.
  4. Décision — si le meilleur score dépasse le seuil calibré empiriquement, la réponse officielle est renvoyée ; sinon, la question est escaladée.

Données : sous-ensemble « Question-réponse » de service-public.fr, via le dataset Hugging Face AgentPublic/service-public (licence Etalab-2.0 — Licence Ouverte).
