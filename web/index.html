Comment ça marche

En 3 étapes simples

  1. Comparaison : votre question est comparée à des milliers de questions déjà traitées par service-public.fr.
  2. Sélection : les 5 questions officielles qui ressemblent le plus à la vôtre sont retenues.
  3. Classement : ces 5 candidates sont réordonnées pour ne garder que la plus pertinente.
  4. Décision : si la correspondance est assez fiable, vous recevez directement la réponse officielle ; sinon, votre question est transmise à un agent humain.

Les réponses proviennent de service-public.fr (données publiques, licence Etalab-2.0, Licence Ouverte).
